# Explore the Universe of Finance

Start here. 29 categories across 7 sectors, 3 country deep dives, and a suite of cross-cutting analyses — all measuring the same thing: how many financial transactions happen on Earth every second.

## The Numbers

| Metric | Value | What it measures |
|:-------|:------|:-----------------|
| **The Big Number** | **~73,750 TPS** | De-duplicated global financial transactions per second (~2.3T/year) |
| **The MEST Number** | **~545,000/s** | Bilateral state changes the financial system actually performs (7.4x multiplier) |
| **Coordinated Peak** | **~147,000--246,000 TPS** | Flash crash on a busy quarter-end |
